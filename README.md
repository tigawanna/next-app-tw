# Pocketbook


![Gopher reading book](https://github.com/tigawanna/next-app-tw/raw/fa6f252cfcb6bdd03c3d7c25878d32f299d8803e/public/pocketbook.jpg)
Minimal social site built with

- [Nextjs13 appDir: React framework](https://nextjs.org/)
- [`pocketbase`: Backend](https://pocketbase.io/)
- [`shadcnui`: components](https://ui.shadcn.com/)
- [`tailwindcss`: Styling ](https://tailwindcss.com/)
- [`@tanstack/query`: For infinite scrolling](https://tanstack.com/query/latest) 
- [`shadcn-helpers`: Custom tools to help improve the  `shadcn/ui` experience](https://github.com/shadcn/shadcn-helpers)
